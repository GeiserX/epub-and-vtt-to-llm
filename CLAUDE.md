# CLAUDE.md — epub-and-vtt-to-llm

## Overview
Pipeline for training and inferring LLMs from ePub books and VTT subtitle files. Extracts, preprocesses, and feeds textual content into language model fine-tuning workflows.

## Tech Stack
- Python
- PyTorch (CUDA 12.4)
- Transformers (HuggingFace)
- Unsloth (efficient fine-tuning)
- TRL (training reinforcement learning)
- LangChain + LangChain OpenAI
- BeautifulSoup4, lxml (ePub parsing)
- webvtt-py (VTT parsing)
- datasets (HuggingFace)

## Development
```bash
pip install -r requirements.txt
python src/process_epub.py
python src/process_vtt.py
python src/transform_text_to_questions.py
python src/train_llama_3_2-3B-Instruct.py
```

Requires CUDA-capable GPU for training.

## Architecture
- `src/process_epub.py` — ePub text extraction
- `src/process_vtt.py` — VTT subtitle extraction
- `src/transform_text_to_questions.py` — text-to-QA pair transformation
- `src/train_flan-t5-large.py` — Flan-T5 fine-tuning script
- `src/train_llama_3_2-3B-Instruct.py` — Llama 3.2 fine-tuning script
- `src/train_qwen-2_5-1_5B-Instruct.py` — Qwen 2.5 fine-tuning script

## Key Rules
- Requires NVIDIA GPU with CUDA 12.4 support
- Never commit model weights or training data
- Multiple model targets supported (Flan-T5, Llama 3.2, Qwen 2.5)
- Input data formats: `.epub` and `.vtt` only
